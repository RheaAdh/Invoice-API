# Invoice API

## To run server `npm i && npm start`in the current directory

### Database credentials

-   Username -> root
-   Password -> password

### Creation Routes

-   GET `http://localhost:5000/createdb`

To Create Tables

-   GET `http://localhost:5000/store/createstoretable`
-   GET `http://localhost:5000/invoice/createinvoicetable`

### Tech stack

-   Nodejs
-   Mysql
